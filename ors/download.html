Instalacija i podešavanje virtuelne mašine

Jednostavno uputstvo za kreiranje i osnovno podešavanje VM-a

1. Preuzimanje potrebnih fajlova

  1. Preuzmi i instaliraj Oracle VM VirtualBox program sa zvaničnog sajta:
  2. https://www.virtualbox.org/wiki/Downloads
  3. Preuzmi Windows 10 ISO fajl sa Microsoft sajta:
  4. https://www.microsoft.com/software-download/windows10
  5. Nakon preuzimanja instaliraj VirtualBox kao običan program.

2. Kreiranje virtuelne mašine

  1. Pokreni VirtualBox program.
  2. Klikni na dugme New.
  3. Unesi naziv virtuelne mašine, na primer Windows 10.
  4. U delu Type izaberi Microsoft Windows.
  5. U delu Version izaberi Windows 10 (64-bit).
  6. Podesi količinu RAM memorije. Preporučeno je najmanje 4096 MB.
  7. Izaberi opciju Create a virtual hard disk now.
  8. Klikni na Create.
  9. Izaberi format diska VDI.
  10. Izaberi Dynamically allocated.
  11. Podesi veličinu diska na najmanje 50 GB.
  12. Ponovo klikni na Create.

3. Dodavanje Windows 10 ISO fajla

  1. U VirtualBox programu izaberi napravljenu virtuelnu mašinu.
  2. Klikni na Settings.
  3. Otvori karticu Storage.
  4. U delu Controller: IDE klikni na ikonicu diska.
  5. Klikni na Choose a Disk File.
  6. Pronađi i izaberi Windows 10 ISO fajl koji si preuzeo.
  7. Klikni na OK da sačuvaš podešavanja.

4. Instalacija Windows 10 sistema

  1. Klikni na dugme Start da pokreneš virtuelnu mašinu.
  2. Sačekaj da se učita Windows instalacija.
  3. Izaberi jezik i tastaturu pa klikni na Next.
  4. Klikni na Install now.
  5. Ako traži ključ, možeš kliknuti na I don't have a product key.
  6. Izaberi verziju Windowsa i klikni na Next.
  7. Prihvati uslove korišćenja.
  8. Izaberi opciju Custom: Install Windows only.
  9. Izaberi virtuelni disk i klikni na Next.
  10. Sačekaj da se instalacija završi i da se sistem restartuje.

5. Dodavanje dodatnog virtuelnog diska

  1. Ugasi virtuelnu mašinu.
  2. Izaberi virtuelnu mašinu i klikni na Settings.
  3. Otvori karticu Storage.
  4. Klikni na Add Hard Disk.
  5. Izaberi Create Disk Image.
  6. Izaberi format VDI.
  7. Podesi veličinu dodatnog diska.
  8. Klikni na Finish, zatim Choose i OK.
